Intro to HTML

HTML is the standard language for creating Web pages. HTML stands for Hyper Text Markup Language. HTML describes the structure of a Web page. It consists of a series of elements. These elements tell the browser how to display the content. HTML elements label pieces of content such as heading, paragraph, link, etc.

HTML

To begin writing an HTML document you must declare the DOCTYPE tag and then the HTML tag.
<!DOCTYPE html>
The DOCTYPE html declaration defines that this document is an HTML document.
The html element is the root element of an HTML page.
<html>

Head

The Head Tag is the element that contains meta information about the HTML page.
<head>
Within the head element an HTML document will also contain a title element.
<title>
This specifies a title for the HTML page which is shown in the browser's title bar.

Body

The Body Tag is the element that defines the document's body.
<body>
It is a container for all the visible contents such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Elements

HTML Elements are defined by a start tag, some content, and then an end tag. Following are some examples of HTML elements:

  • The Header Tag defines a heading for page content.
    <header>

  • The Section Tag defines a section for page content.
    <section>

  • The P Tag defines a paragraph for page content.
    <p>

    • Attributes

    HTML Attributes provide additional information for the HTML element to which they are assigned.

  • The ID attribute is a unique identifer for the specified element.
    <id="">

  • The Class attribute is an identifier that can be used for multiple elements within the content.
    <class="">

    • Comments

    HTML comments are not displayed in the browser, but they can help document your HTML source code. With comments you can place notifications and reminders in your HTML code.
    <!-- This is a comment. -->

    Conclusion

    Now that you've learned HTML, you can explore styling your webpages with CSS, Cascading Style Sheets!
